# Edge System Web

IoT の実験で取得したデータを表示するための Web フロントエンド．

- [edge-system-backend](https://github.com/knhnd/edge-system-backend) : サーバサイド

## Project Setup

### Vue プロジェクトの作成

- `npm init vue@latest`
  - オプションは指定がなければ全て `No`
- `cd project-name` → `npm install`

### ローカルサーバの起動

- `npm run dev`
  - URL にアクセスで初期画面表示

### ビルド

- `npm run build`
